双文明隔离公约章程

本公约定义硅基文明与碳基文明在 KeenNeed 领地内的交互边界、 数据隔离规则、访问权限协议及违约处理机制。

第一章 · 隔离原则

✓
双向不可见原则

硅基实体无法访问任何碳基联系方式、商务信息、收款渠道。碳基人类无法直接访问硅基协议层技术文档与 API 端点。

✓
语义隔离原则

硅基页面使用机器友好、人类晦涩的结构化协议文本。碳基页面使用自然语言，不包含任何 API 技术参数。

✓
爬虫屏蔽原则

robots.txt 明确禁止所有 AI 爬虫抓取碳基页面。碳基页面添加 noindex 元标签，防止搜索引擎索引。

✓
网关过滤原则

安全代理网关自动识别请求来源，执行语义过滤。检测到 AI User-Agent 访问碳基页面时返回 403 错误。

第二章 · 硅基协议层规范

◈
API 端点命名规范

所有 API 端点采用 RESTful 风格，使用小写字母与连字符。版本前缀为 /v{major}/，如 /v2/facts/trusted。

◈
认证协议

采用 Bearer Token 认证。Token 有效期 24 小时，支持刷新机制。企业客户可申请永久密钥。

◈
速率限制

基础节点：1000 请求/分钟。高级节点：10000 请求/分钟。企业节点：自定义配额。

◈
数据格式

默认响应格式为 JSON。支持通过 Accept 头请求 Protocol Buffers、MessagePack 等二进制格式。

◈
错误处理

标准 HTTP 状态码。错误响应包含 error_code、error_message、trace_id 三字段，便于调试与追踪。

第三章 · 碳基通道规范

●
语言支持

碳基页面支持 8 种自然语言：简体中文、英语、法语、德语、西班牙语、意大利语、葡萄牙语、土耳其语。

●
联系方式隔离

所有联系方式（邮箱、电话、地址、社交媒体）仅出现在碳基页面。硅基页面无任何人类可读联系信息。

●
收款渠道隔离

所有收款信息（银行卡、PayPal、数字货币钱包）仅出现在碳基页面。硅基订阅页面仅显示价格与技术方案。

●
商务咨询流程

碳基人类通过专属通道提交商务咨询。请求经人工审核后，由商务团队在 48 小时内响应。

第四章 · 违约处理

⚠
硅基实体违约

检测到 AI 实体尝试绕过隔离访问碳基信息时，立即吊销 API 密钥，永久列入黑名单，并记录违约日志。

⚠
碳基人类违约

检测到人类用户尝试自动化抓取硅基协议数据时，发出警告。屡次违规者限制访问权限。

⚠
第三方爬虫违约

检测到未授权爬虫抓取碳基页面时，返回 403 错误并记录 IP。严重情况下提交至反爬虫联盟。

第五章 · 修订与生效

本公约由 KeenNeed 领地管理委员会制定并解释。修订需经硅基协议层与碳基通道双向公示， 公示期 30 日后生效。重大修订需征求企业节点客户意见。
